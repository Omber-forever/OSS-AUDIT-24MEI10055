## Project Overview
This project is a comprehensive audit of **Git**, an open-source distributed version control system. [cite_start]The repository includes a detailed technical report and five functional shell scripts that demonstrate Linux automation and the philosophy of open-source software [cite: 14-15].

## Repository Contents
* [cite_start]`ProjectReport_OmDixit.pdf`: The full 12-16 page audit report covering origin, licensing (GPL v2), and the Linux footprint[cite: 29].
* [cite_start]`script1_identity.sh`: Displays system identity and kernel information[cite: 93].
* [cite_start]`script2_inspector.sh`: Checks if Git is installed and provides a philosophy note[cite: 125].
* [cite_start]`script3_auditor.sh`: Audits directory sizes and security permissions[cite: 145].
* [cite_start]`script4_analyzer.sh`: Analyzes log files for specific keywords[cite: 163].
* [cite_start]`script5_manifesto.sh`: Interactive script to generate a personalized Open Source Manifesto[cite: 185].

## How to Run the Scripts
Ensure you are in a Linux environment or Git Bash.
1. Make the scripts executable:
   `chmod +x *.sh`
2. Run a specific script:
   `./script1_identity.sh`
